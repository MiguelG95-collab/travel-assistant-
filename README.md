# Travel Assistant Profesional Minimalista

Esta versión conserva el CRM, cotizaciones, proveedores, favoritos, respaldos y buscador de destinos, con una interfaz más limpia y profesional.

## Actualizar la app publicada

1. Descomprime el ZIP.
2. Sustituye `app.py`, `requirements.txt` y `README.md` en GitHub.
3. Haz **Commit changes**.
4. Streamlit actualizará la aplicación automáticamente.
5. Si tarda, usa **Reboot app** desde Streamlit Community Cloud.

## Importante

Descarga respaldos periódicos. El almacenamiento local de Streamlit Community Cloud no debe considerarse permanente.
